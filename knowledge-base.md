### 🏗️ Game Progression Stages

**Early Game (Levels 1-20):**
- Priority: HQ progression over buildings
- Resource targets: Focus on Iron/T9 farming
- Key events: First blood systems, daily trucks/trains
- Hero priority: All-tank squads for 20% bonus

**Mid Game (Levels 20-30):**
- Transition to DPS-focused heroes
- Alliance coordination becomes critical
- Resource management: Diamond saving strategies
- Event focus: SVS preparation and MG optimization

**Late Game (30+):**
- UR hero integration thresholds
- Advanced combat positioning
- Server politics and diplomacy
- Endgame resource optimization

**F2P Viable Strategies:**
- Barracks staggering technique
- Survivor min-maxing for resource buildings
- Efficient event participation patterns

**P2P Spending Priorities:**

- HQ upgrade limited packs
- Drone component investments
- UR hero unlocking through diamonds vs cash
